# Computer Fundamentals

## Overview

This module introduced the foundational concepts of how computers function internally.

Topics explored included:
- Hardware components
- CPUs
- RAM
- Storage
- Input and output devices

## Key Concepts Learned

### CPU
The CPU processes instructions and performs calculations.

### RAM
RAM temporarily stores data currently being used by the computer.

### Storage
Storage devices permanently save files and operating systems.

### Input and Output
Input devices send data to the computer, while output devices display results to the user.

## Reflection

This module provided foundational knowledge about computer hardware and system architecture that supports future learning in cyber security and systems engineering.
