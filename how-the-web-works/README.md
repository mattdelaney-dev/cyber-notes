# How The Web Works

## Overview

This module introduced the fundamentals of how websites and web applications function.

Topics explored included:
- HTTP and HTTPS
- Web servers
- Browsers
- DNS
- Requests and responses

## Key Concepts Learned

### HTTP
HTTP is the protocol used for communication between web browsers and servers.

### HTTPS
HTTPS secures communication using encryption.

### Web Servers
Web servers host and deliver website content to users.

### DNS
DNS converts human-readable domain names into IP addresses.

## Reflection

Understanding how web technologies function is important for both web development and cyber security because many attacks target web applications and network communication.
