# The CIA Triad

## Overview

This room introduced the CIA Triad, one of the foundational models used in cyber security to guide security practices and decision-making.

Topics explored included:
- Confidentiality
- Integrity
- Availability
- Security mindset
- Risk and protection principles

## Key Concepts Learned

The CIA Triad is a security model built around three core principles:
- Confidentiality
- Integrity
- Availability

These principles help organizations design secure systems and protect information.

---

## Confidentiality

Confidentiality ensures that sensitive information is only accessible to authorized individuals.

Examples of confidentiality controls include:
- Passwords
- Encryption
- Multi-factor authentication
- Access control systems

The goal is to prevent unauthorized access to data.

Example:
Only authorized employees should be able to view confidential company information.

---

## Integrity

Integrity ensures that data remains accurate, consistent, and unmodified unless authorized changes are made.

Integrity protections help prevent:
- Data tampering
- Unauthorized modifications
- Corruption of information

Examples include:
- File hashing
- Permissions
- Backups
- Version control systems

Example:
A bank account balance should not be altered without proper authorization.

---

## Availability

Availability ensures that systems, services, and data remain accessible when needed.

Organizations use various methods to improve availability, including:
- Backups
- Redundancy
- Load balancing
- Disaster recovery planning

Availability is important because downtime can disrupt business operations and services.

Example:
A website should remain online and accessible to users.

---

## The Security Mindset

Cyber security professionals must think critically about:
- Risks
- Threats
- Vulnerabilities
- Defensive strategies

A security mindset involves constantly evaluating:
- What could go wrong
- How systems could be attacked
- How risks can be reduced

This mindset is important for both offensive and defensive cyber security roles.

---

## Why This Matters

The CIA Triad is one of the most important foundational concepts in cyber security.

It is used in:
- Security planning
- Risk management
- System design
- Policy development
- Incident response

Understanding the CIA Triad helps build a strong foundation for future cyber security learning.
