# JavaScript: Simple Demo

## Overview

This room introduced the basic structure of JavaScript programs and explored core programming concepts used in web development and interactive applications.

Topics explored included:
- Variables
- User input
- Conditional statements
- Iterations and loops
- Basic JavaScript syntax

## Key Concepts Learned

### Variables

Variables are used to store and manage data within a program.

Examples of stored data include:
- Numbers
- Text
- Boolean values

Example:

```javascript
let name = "Matthew";
let age = 19;
```

Variables allow programs to dynamically store and manipulate information.

---

## Prompting User Input

JavaScript can interact with users by requesting input.

Example:

```javascript
let username = prompt("Enter your name:");
```

User input allows programs to become interactive and respond dynamically to different values.

---

## Conditional Statements

Conditional statements allow programs to make decisions based on conditions.

Common statements include:
- if
- else if
- else

Example:

```javascript
let age = 18;

if (age >= 18) {
    console.log("Adult");
} else {
    console.log("Minor");
}
```

Conditional logic controls how programs respond to different situations.

---

## Iterations and Loops

Loops allow programs to repeat tasks efficiently.

Common loop types include:
- for loops
- while loops

Example:

```javascript
for (let i = 0; i < 5; i++) {
    console.log(i);
}
```

Loops are important for automation and repetitive processing.
