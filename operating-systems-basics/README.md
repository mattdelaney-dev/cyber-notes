# Operating Systems Basics

## Overview

This module explored the role of operating systems and how they interact with computer hardware and applications.

Topics explored included:
- Linux basics
- File systems
- Processes
- Permissions
- Command-line usage

## Key Concepts Learned

### Operating Systems
An operating system manages hardware resources and provides services for applications.

### Linux
Linux is widely used in servers, cloud systems, and cyber security environments.

### File Permissions
Permissions control access to files and directories.

### Processes
Processes are running instances of programs managed by the operating system.

## Commands Learned

- pwd
- ls
- cd
- mkdir
- touch

## Reflection

This module strengthened foundational Linux and operating system knowledge that is important for cyber security and systems administration.
