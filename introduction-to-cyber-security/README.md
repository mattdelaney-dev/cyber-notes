# Introduction to Cyber Security

## Overview

This module introduced foundational cyber security concepts, including both offensive and defensive security principles.

Topics explored included:
- Threats and vulnerabilities
- Basic penetration testing concepts
- Defensive security measures
- Online safety and security awareness
- Ethical hacking fundamentals

## Key Concepts Learned

### Offensive Security
Offensive security focuses on identifying and exploiting vulnerabilities in systems to improve overall security.

### Defensive Security
Defensive security involves protecting systems, monitoring threats, and reducing vulnerabilities.

### Vulnerabilities
A vulnerability is a weakness in a system that can potentially be exploited by an attacker.

## Tools and Technologies Mentioned

- Web applications
- Basic security tooling
- Authentication systems

## Reflection

This module provided a strong introduction to the cyber security field and helped build an understanding of how attackers and defenders interact in real-world environments.
