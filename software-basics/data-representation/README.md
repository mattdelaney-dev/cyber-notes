# Data Representation

## Overview

This room introduced how computers represent colors and numbers using binary and hexadecimal systems.

Topics explored included:
- Binary numbers
- Hexadecimal numbers
- Color representation
- Decimal conversion
- Computer data storage basics

## Learning Objectives

- Representing 8 colors
- Representing 16 million colors
- Binary numbers
- Hexadecimal numbers
- Octal numbers (optional)

## Key Concepts Learned

### Binary Numbers

Computers use binary digits (bits) to represent data using only:
- 0
- 1

Binary is the foundation of how computers process and store information.

Example:

```txt
1010
```

### Decimal System

Humans commonly use the decimal system (base 10), which uses digits from 0 to 9.

Examples:
- 17
- 99
- 239
- 3049

### Hexadecimal System

Hexadecimal (base 16) is commonly used in computing because it provides a shorter and more readable representation of binary values.

Hexadecimal uses:
- 0-9
- A-F

Example:

```txt
FF
```

### Color Representation

Computers represent colors using combinations of:
- Red
- Green
- Blue (RGB)

Binary and hexadecimal values can be combined to create millions of colors.

Examples:

```txt
#FF0000 = Red
#00FF00 = Green
#0000FF = Blue
```

### Data Storage

Computers store and process all forms of information using binary values.

This includes:
- Numbers
- Images
- Colors
- Text
- Videos

## Why This Matters

Understanding binary and hexadecimal systems is important in cyber security and computer science because:
- Memory addresses are often displayed in hexadecimal
- Network and system tools use hexadecimal values
- Binary is the foundation of all digital systems
- Low-level computing concepts rely heavily on binary representation

## Reflection

This room provided foundational knowledge about how computers internally represent and process data using binary and hexadecimal systems. Understanding these concepts is essential for future learning in networking, operating systems, programming, and cyber security.
