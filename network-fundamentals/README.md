# Network Fundamentals

## Overview

This module introduced the core concepts of computer networking and communication between devices.

Topics explored included:
- IP addresses
- DNS
- Routers and switches
- TCP/IP
- Packets
- Ports and protocols

## Key Concepts Learned

### IP Addresses
IP addresses identify devices on a network and allow communication between systems.

### DNS
DNS translates domain names into IP addresses.

### TCP/IP
TCP/IP is the core communication protocol suite used on modern networks.

### Ports
Ports allow multiple services to operate on a single device.

Examples:
- Port 80 → HTTP
- Port 443 → HTTPS

## Reflection

Understanding networking fundamentals is essential for cyber security because attacks, monitoring, and defensive systems all rely heavily on network communication.
