## Key Concepts Learned

### Variables

Variables are used to store data in a program.

Examples of stored data include:
- Numbers
- Text
- Boolean values

Example:

```python
name = "Matthew"
age = 19
```

Variables allow programs to store and manipulate information dynamically.

---

## Conditional Statements

Conditional statements allow programs to make decisions based on conditions.

Python commonly uses:
- if
- elif
- else

Example:

```python
age = 18

if age >= 18:
    print("Adult")
else:
    print("Minor")
```

Conditional logic is essential for controlling program behavior.

---

## Iterations and Loops

Loops allow programs to repeat actions multiple times.

Common loop types include:
- for loops
- while loops

Example:

```python
for number in range(5):
    print(number)
```

Loops improve efficiency by automating repetitive tasks.
